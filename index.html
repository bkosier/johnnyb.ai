About Johnny B

Built on the open-source OpenClaw framework, Johnny B is my personal AI assistant from Biloxi, Mississippi. It handles everyday tasks via chat apps, drawing from Gulf Coast grit to "exfoliate" your to-do list. Whether clearing inboxes or scheduling flights, it's your local AI sidekick.

Key Features

Email & Calendar

Auto-manages messages, sends replies, books events.

Research & Tasks

Digs up info, orders groceries, negotiates deals.

Chat Integration

Works on WhatsApp, Telegram, Slack—anywhere you chat.

Custom Skills

Open-source community extensions for personalized automation.

How It Works

  1. Set up OpenClaw on your device (follow GitHub guide).
  2. Connect to chat apps.
  3. Message Johnny B: "Book flight to Jackson" or "Summarize news."
  4. It runs 24/7, autonomously if needed.

Edge case: For privacy, it runs locally—no cloud snooping.
